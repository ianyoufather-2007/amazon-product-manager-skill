# Roadmap

This project is early-stage and will be improved through small, visible releases.

## Week 1: Make The Skill Easy To Try

- Add a Chinese README section.
- Add one complete market-screening sample using public or synthetic data.
- Add one stop-loss sample using public or synthetic data.
- Improve install instructions for Windows users.

## Week 2: Improve Decision Quality

- Expand the market screening scorecard.
- Add clearer thresholds for `Enter Market`, `Watch Market`, and `Avoid Market`.
- Add a profit rescue checklist for weak margin, high ACoS, and return drag.
- Add more examples for small-brand constraints.

## Week 3: Add Public Test Cases

- Add 3 anonymized example prompts and expected output structures.
- Add a contribution checklist for new decision templates.
- Add issue templates for bug reports and framework improvements.

## Week 4: Prepare Program Applications

- Review docs, examples, and repository hygiene.
- Open and close a few real maintenance issues.
- Prepare an updated open-source program application note.
- Ask for feedback from sellers, AI workflow builders, and Codex users.

## Ongoing

- Keep examples free of credentials, private seller data, supplier quotes, paid-tool exports, and browser sessions.
- Prefer small commits that improve one workflow or template at a time.
- Keep `SKILL.md` concise and move detailed frameworks into `references/`.
