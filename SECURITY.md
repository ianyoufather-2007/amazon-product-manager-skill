# Security

Please do not open public issues containing credentials, paid-tool exports, customer data, supplier quotes, browser sessions, cookies, or other private business information.

If you find a security or privacy issue in this repository, report it privately to the repository maintainer.

## Sensitive Data To Avoid

- `.env` files
- Browser profiles and session folders
- Seller account credentials
- Amazon order, customer, or advertising exports
- Supplier contact details and quotes
- Proprietary paid-tool datasets
- Screenshots containing account, vendor, or customer information
