# Code of Conduct

This project welcomes practical, respectful collaboration.

## Expected Behavior

- Be constructive and specific when giving feedback.
- Assume good intent, but make room for correction.
- Keep discussions focused on Amazon product-management workflows, examples, documentation, and skill behavior.
- Respect privacy: do not post private seller data, account information, raw marketplace exports, screenshots, supplier quotes, customer records, or credentials.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private business data or personal information without permission.
- Repeated off-topic promotion, spam, or misleading claims.
- Pressuring maintainers or contributors to reveal private data, paid-tool exports, or account access.

## Reporting

If you see a problem, open a GitHub issue with a minimal description that does not expose private information. If a public issue would reveal sensitive data, describe the category of concern instead of the underlying private content.

Maintainers may edit, hide, or remove content that exposes private information or disrupts productive collaboration.
